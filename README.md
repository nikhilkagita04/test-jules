# The Grind Diary - Project Overview

This repository contains two main components for "The Grind Diary":

1.  A **static HTML landing page concept** (described below).
2.  The main **interactive React application** (located in the `grind-diary/` subdirectory).

---

## Static Landing Page Details

This section describes the files for the static HTML landing page concept of "The Grind Diary". It's a simple website built with HTML, CSS, and a small amount of JavaScript, intended for demonstration or as a basic entry point.

### How to View the Static Page

To view the static landing page:

1.  Clone or download the repository/files.
2.  Navigate to the project's root directory.
3.  Open the `index.html` file directly in your preferred web browser (e.g., Chrome, Firefox, Safari, Edge).

No special build steps or local server setup is required for the static page.

### Static Page Project Files

*   `index.html`: The main HTML structure of the landing page.
*   `style.css`: Contains all the styles for the landing page.
*   `script.js`: Includes basic JavaScript for functionalities like smooth scrolling (if implemented).
