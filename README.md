# The Grind Diary - Landing Page

This project contains the files for the landing page of "The Grind Diary". 
It's a static website built with HTML, CSS, and a small amount of JavaScript.

## How to View

To view the landing page:

1.  Clone or download the repository/files.
2.  Navigate to the project directory.
3.  Open the `index.html` file directly in your preferred web browser (e.g., Chrome, Firefox, Safari, Edge).

No special build steps or local server setup is required as this is a simple static site.

## Project Files

*   `index.html`: The main HTML structure of the landing page.
*   `style.css`: Contains all the styles for the landing page.
*   `script.js`: Includes basic JavaScript for functionalities like smooth scrolling.
